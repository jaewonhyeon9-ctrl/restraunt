# 식당 운영 현황 설문 — Google Forms용 스크립트

> 작성일: 2026-05-08
> 목적: ① 시장조사 (현재 사장님들이 운영을 어떻게 처리하는지) ② 베타 후보 발굴
> 활용: Google Forms / 네이버 폼에 그대로 복사·붙여넣기
> 예상 응답 시간: 5분

---

## 📌 사용 방법

1. **Google Forms** (https://forms.google.com) 접속
2. "+ 빈 양식" 클릭
3. 아래 제목·설명·각 섹션·질문을 **순서대로 복사·붙여넣기**
4. "응답" 탭에서 결과 확인 (스프레드시트 연결 추천)

⏱ 만드는 데 10~15분이면 완성.

---

## 📋 양식 제목

```
식당 사장님 운영 현황 설문 (5분 / 응답 시 베타 우선 초대)
```

## 📋 양식 설명 (제목 아래)

```
안녕하세요. 작은 식당을 직접 운영하면서 식당 운영 자동화 앱(오토드림)을 만들고 있는 1인 개발자입니다.

영업 중 매출·재고·직원 관리에 시간이 너무 많이 들어 직접 도구를 만들었고, 다른 사장님들은 어떻게 처리하시는지 궁금해서 설문 드립니다.

■ 응답 시간: 약 5분
■ 익명 응답 (마지막 연락처는 선택)
■ 응답해주신 사장님께는 정식 출시 후 평생 무료 베타 자격 우선 안내

답변 주신 데이터는 앱 개선과 정부지원 사업 기획에만 사용되며, 외부에 공유되지 않습니다.

감사합니다.
```

## 💡 중요 팁

1. **Google Forms 만든 후 단축 URL 만들기**
   - bit.ly · t.ly · 네이버 me2.do 등으로 짧게

## 6. 운영자가 점수 빠르게 계산하는 스프레드시트 (Google Sheets)

응답 50건 이상 받으면 수동 계산 부담. 스프레드시트로 자동화:

```
A열: 응답 ID (Google Forms 자동)
B열: 응답 타임스탬프
C~U열: 각 질문 응답 텍스트
V열~AE열: 각 응답에 대한 점수 (VLOOKUP으로 채점표 조회)
AF열: 총점 (=SUM(V:AE))
AG열: 진단 결과 (=IFS(AF2<=20,"마스터", AF2<=40,"평균", AF2<=60,"수기의존", AF2<=80,"위험", "긴급"))
AH열: 추천 메시지 자동 생성
```

→ 응답 들어오면 자동으로 진단 결과 컬럼 채워짐. 1분 이내 답장 가능.

스프레드시트 템플릿이 필요하면 별도 요청 주세요.
