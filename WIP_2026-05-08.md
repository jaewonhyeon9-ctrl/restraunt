## 🎯 사용자 워크플로우 (오늘 작업으로 가능)

### 1. 신규 재고 등록
1. 재고 페이지 → "+ 품목"
2. 단위 유형 chip 선택 (무게/용량/갯수)
3. 사이즈/단위/가격 입력 → 자동 환산 미리보기 (1단위당 + 1g당)
4. 재료 분류 chip 선택 OR 🤖 AI 자동분류 클릭
5. 저장

### 2. 레거시 재고 1g당 가격 자동 설정
1. 재고 페이지 → 보라색 배너 "🪄 1g당 가격 자동 설정 가능"
2. "일괄 설정" 버튼 → 이름에서 감지된 모든 품목 한 번에 환산
3. 감지 안 된 품목은 편집 모달에서 직접 입력

### 3. 메뉴 레시피 등록 (g/ml 단위)
1. 메뉴 페이지 → 메뉴 카드 탭 → 레시피 모달
2. "+ 재료 추가" → 재고 검색 → 클릭
3. 1메뉴당 사용량 g/ml로 입력 → 원가 자동 계산
4. 패키지 중량 미설정 품목은 🪄 자동 감지 버튼 표시
