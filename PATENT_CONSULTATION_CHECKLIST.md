# Part 2 — 공익변리사 무료 출원 절차

## I. 공익변리사 특허상담센터란?

**한국지식재산보호원**이 운영. 자격 충족 시 **변리사 수임료 면제**, 출원료만 부담.

- 홈페이지: https://www.pcc.or.kr (공익변리사 특허상담센터)
- 전화: 02-6006-4300
- 운영: 한국지식재산보호원 (KIPRIPRO)

### 자격 기준 (둘 중 하나)
1. **소득 기준**: 기초생활수급자 / 차상위계층 / 한부모가족 / 다문화가족 / 국가유공자 등
2. **사업 기준**: 사회적기업·협동조합·1인 창업자(소득 기준 충족 시) 등

→ 일반 소상공인은 일부만 해당. 사용자분 자격 확인 필요.

### 자격 확인 방법
1. 홈페이지 "이용대상" 메뉴에서 직접 확인
2. 또는 전화 02-6006-4300으로 본인 상황 설명 후 자격 문의

## II. 자격 미달 시 대안 (대부분 더 유리할 수도)

### 1. 지역지식재산센터 (한국발명진흥회 KIPA)
- 검색: "지역지식재산센터 + [본인 거주지]" (예: "경기지식재산센터", "서울지식재산센터")
- **소상공인 누구나 무료 이용** (소득 기준 없음)
- 무료 변리사 상담 + 선행기술 조사 + 명세서 작성 가이드
- 출원 비용 보조사업도 운영

### 2. 창업진흥원 IP 바우처
- 창업 7년 이내 기업 대상
- IP 출원·등록·분쟁 비용 지원 (수백만원~)
- 사이트: K-Startup (k-startup.go.kr)

### 3. 특허청 출원료 감면
- 개인·소상공인: **70% 감면** (자동)
- 1건 출원료 약 **15만원 → 4만5천원 수준**

### 4. 모두의 창업 지원금 (이미 신청)
- 결과 발표 시 자금 일부를 IP 비용으로 활용

## III. 추천 경로 (사용자분 케이스)

```
1단계: 오늘 변리사 상담
   └─ 청구 후보 정리 + 비용 견적 + 정부지원 안내 받기

2단계: 자격 확인 (이번 주)
   ├─ 공익변리사 (소득 기준 충족 시)
   ├─ 지역지식재산센터 (소상공인이면 누구나 OK)
   └─ 창업진흥원 IP 바우처 (창업기업 신고 후 가능)

3단계: 우선 1건 출원 (이번 달 ~ 다음 달)
   └─ 1순위 청구 #9 (메뉴 원가율) 부터

4단계: 상표 동시 진행 (이번 달)
   └─ 제9·35·42류 출원 → 등록 약 12개월

5단계: 모두의 창업 결과 후 (1~2개월)
   └─ 지원금 받으면 추가 출원 (#7 AI 리포트, #10+12 묶음)
```

## IV. 오늘 변리사 상담 마지막 질문

> "저는 본인 식당 운영하면서 앱 개발하는 1인 창업자입니다.
> **공익변리사 / 지역지식재산센터 / IP 바우처** 중 어떤 경로가 적합할까요?
> 직접 신청하면 되는지, 아니면 변리사님께서 신청까지 도와주시는지 알려주세요."
