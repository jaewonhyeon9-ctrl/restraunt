# 오토드림 — 특허 출원 1페이지 요약

> 변리사 / 투자자 / 의사결정자용 / A4 1매
> 최종 갱신: 2026-05-06 (청구 후보 13개로 확대 — 사진 인증 체크리스트, 단가 정규화 추가)

---

## 발명 명칭
**식당 운영 데이터를 OCR 기반으로 자동 수집·정규화하여 매출 및 재고를 통합 관리하는 PWA 시스템 및 그 방법**

## 한 문장 정의
영수증 한 장만 찍으면 매출 등록 + 재고 자동 차감 + 발주 알림이 동시에 처리되는, 한국 식당 환경(POS 다양성·영세 IT 환경)에 특화된 PWA 솔루션.

---

## 핵심 청구 요약 (Top 5)

**① OCR 기반 식당 매출·재고 자동화 시스템 (메인 청구)**
- 멀티모달 LLM이 영수증/POS 화면에서 메뉴·수량·금액을 구조화 JSON으로 추출
- 메뉴-재고 매핑 사전을 참조해 단일 트랜잭션으로 매출+재고+로그 동시 처리
- 차별점: 매출-재고 분리 운영하는 종래 시스템 대비 **시간 70%↓, 인적 오류 90%↓**

**② 다단계 폴백 매출 입력 방법**
- 단일 화면에서 직접/엑셀/CSV/OCR/외부 API의 5가지 입력 경로 동시 제공
- 컬럼명 자동 감지(한국어/영어 사전) + 데이터 타입 추론
- 차별점: POS 종류·식당주 IT 숙련도와 **무관하게 100% 입력 가능**

**③ AI 매출 자연어 리포트 자동 생성 방법**
- 시계열 통계 + 이상치 탐지 + 도메인 컨텍스트 → LLM 진단형 리포트
- "왜 어제 매출이 떨어졌는지" 같은 자연어 분석
- 차별점: 그래프만 제공하는 종래 SaaS 대비 **의사결정 즉시 활용 가능**

**④ GPS 정확도 인지 출퇴근 검증 + 다중측정 (NEW 2026-05)**
- 단순 거리 비교가 아닌 `허용 거리 = 반경 + min(GPS 정확도, 200m)` 동적 검증
- 3초간 watchPosition 다중 측정 후 가장 정확한 값 자동 선택, ±10m 이하 즉시 종료
- 직원 거부 시 사장에게 즉시 푸시 → 매장별 GPS 검증 우회 토글로 1탭 해결
- 차별점: 실내·도심 빌딩의 GPS 오차로 무용지물이던 종래 GPS 출퇴근의 **실사용 가능성 확보**

**⑤ 사진 인증 퇴근 필수 체크리스트 (NEW 2026-05)**
- 사장이 항목별로 "사진 인증 필수" / "퇴근 시 필수" 두 플래그 독립 설정
- 직원 측: 사진 미첨부 시 서버측 체크 거부, 자동 압축 후 CDN 업로드
- 퇴근 클릭 시 미완료 필수 항목 자동 차단 + 안내 → 마감 누락 0건화
- 차별점: 종래 체크리스트 SaaS의 단순 "체크박스 + 메모"를 **증빙 강제 + 흐름 차단**으로 진화

## 종래 기술 대비 핵심 차별점

| 항목 | 종래 (캐셔플/페이히어 등) | 오토드림 |
|---|---|---|
| 매출 입력 | POS 단말 직접 연동만 | **5가지 경로 동시** |
| 영수증 OCR | 미지원 또는 텍스트만 | **멀티모달 LLM 구조화 추출** |
| 재고 차감 | 매출과 분리, 별도 입력 | **단일 트랜잭션 자동** |
| 재고 가격 비교 | 없음 또는 단순 단가 | **거래처/제조사/10g당 정규화 비교** |
| 단말 | POS 단말 + 데스크톱 | **휴대폰 PWA만** |
| AI 리포트 | 없음 (그래프만) | **자연어 진단형** |
| GPS 출퇴근 | 단순 거리 검증, 실내 사용 불가 | **정확도 인지 + 다중측정 + 사장 우회 푸시** |
| 체크리스트 | 단순 체크박스 | **사진 인증 강제 + 퇴근 차단** |
| 다점포 | 매장별 별도 계정/별도 결제 | **단일 계정 매장 전환 + 매장별 설정 진행률 추적** |
| 비용 | 월 22,000~33,000원 | **무료 (전 기능)** |

## 등록 가능성 (자체 평가)

- **매우 유망 (출원 1건 묶음 권장)**: ② OCR 매출 자동 입력 + ③ OCR→재고 차감 + ④ 메뉴-재고 매핑 학습
- **유망 (별도 출원)**: ⑦ AI 자연어 리포트
- **신규·유망 (별도 출원 검토)**: ⑩ GPS 정확도 인지 출퇴근 검증 + 다중측정 + 우회 푸시 (실내 GPS 한계 극복은 종래 솔루션이 다루지 않은 영역)
- **신규·유망 (별도 출원 검토)**: ⑫ 사진 인증 퇴근 필수 체크리스트 (증빙 강제 + 흐름 차단의 결합은 종래 단순 체크리스트와 차별)
- **유망 (단일 출원 또는 ① 종속항)**: ⑬ 거래처/제조사/10g당가격 정규화 비교 (이질 단위·포장의 단가 비교 정규화)
- 나머지: 명세서 본문에 포함
